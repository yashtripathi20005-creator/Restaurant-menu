# 🍽️ Restaurant Menu App

A full-featured restaurant menu web application built with Python Flask.

## Features
- 📋 Browse menu by categories
- 🔍 Search dishes by name/description
- 🛒 Add items to cart
- 🗑️ Manage cart (remove/clear)
- 🌶️ Spicy & Vegetarian tags
- 💳 Checkout simulation
- 📱 Fully responsive

## Tech Stack
- **Backend:** Python Flask
- **Frontend:** HTML5, CSS3, JavaScript
- **API:** RESTful JSON endpoints

## Installation

```bash
# Clone repository
git clone <your-repo-url>
cd restaurant-menu

# Install dependencies
pip install -r requirements.txt

# Run application
python app.py
